🚀 HOW TO DEPLOY THIS APPLICATION

1. RIGHT-CLICK on "DEPLOY.bat"
2. SELECT "Run as administrator"  
3. WAIT for it to complete
4. FOLLOW the IIS setup instructions shown

That's it! ✅

The script automatically:
- Fixes any GitHub download issues
- Builds the application
- Deploys to C:\inetpub\wwwroot\simple-iis-app\
- Shows you the next steps for IIS setup

🔧 Requirements:
- .NET 9.0 SDK installed
- IIS with ASP.NET Core Module V2
- Run as Administrator

📁 Files deployed to: C:\inetpub\wwwroot\simple-iis-app\
🌐 Website URL: http://localhost:8080 (after IIS setup)
